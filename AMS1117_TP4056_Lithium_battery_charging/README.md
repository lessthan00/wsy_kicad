### 状态指示说明

TP4056有两个漏极开路状态指示输出端CHRG和STDBY。当充电器处于充电状态时，CHRG被拉到低电平，在其他状态
CHRG为高阻态；当电池充电结束后，STDBY被拉到低电平，在其他状态STDBY为高阻态。
当电池没有接到充电器时，CHRG闪烁表示没有安装电池。
